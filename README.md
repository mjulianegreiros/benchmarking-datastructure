# benchmarking-datastructure
Sistema de Benchmarking desenvolvido para a disciplina de Algoritmos e Complexidade Computacional na Universidade Federal do Ceará.
Visando conhecer mais profundamente algumas das estruturas de dados avançadas, esse trabalho tem como objetivo fazer uma análise comparativa entre: 
  1. Árvore binária de busca: estrutura baseada em nós, onde cada nó a esquerda tem valor menor que a raiz e cada nó a direita tem valor maior que a raiz.
  2. Árvore AVL: estrutura que segue a mesma lógica das árvores binárias de busca, entretanto, garante que a diferença entre alturas de sub-árvores à esquerda e à direita do nó raiz seja, no máximo e em módulo, 1.
  3. Tabela hash: estrutura em que a posição de determinado elemento no vetor baseia-se no retorno de uma função. 
Essa análise foi feita através da implementação das estruturas de dados através da linguagem C de programação e uso de bibliotecas que analisam o tempo de execução do programa, como a time.h.
